Java Servelets
are basically used as an application for interfaces and also make out of the possibilities of Servers on incoming messages that will respond to any kind of request.

____________________________________________________________________________________

requestDispatcher
it provides the instruments of using the request to other resources like html, servelet, jsp.

Two Methods
-public void forward() - forwards a request from a servelet to a resource on the server
-public void include() - includes the content of a resource

____________________________________________________________________________________

sendRedirect
it is used to redirect responses to other resources. This method is used by the client and is always sends a new request then it can be used in and out of the server.

Syntax:
public void sendRedirect(URL) throws IOException
response.sendRedirect("http://www.yahoo.com");

____________________________________________________________________________________

Specifying Errors
will be able to control the error that will be appearing in the web client

error-code
It is optional and used as a valid error code

exception-type
It is optional and use just like java exception type as an exception in HTTP.

location
It is used to redirect a page to a different resource

Different types of Error that can occur in servelets
-ServletException
-IOException
-RuntimeException

____________________________________________________________________________________

Servelet LifeCycle

-instantiation -> constructor call
creates an instance of the servelet to service client requests

-initialization -> init () method
invoked only once and is intended for any startup initialization code for the servelet

-request handling -> service () method
invoked for each client request. servelet performs whatever logic necessary in order to serve and generate a response

-destruction -> destroy () methodd
invoked before the servelet instance is unloaded and is intended

____________________________________________________________________________________

HttpServlet
-Java Servelet used to handle Http Requests and generate Http Responses.
-Hosted in a web container which is a component in a java application server.

____________________________________________________________________________________

HttpServlet Response
-object representation of the Http response generated by the servelet and and sent back to the requesting client
-the web container facilitates the conversion to and from the Http request/response to Http ServeletResponse/Request
